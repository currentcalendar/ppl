# Current Calendar

El proyecto se divide en dos principales servicios: **API** y **cliente**

## Levantar la API

Es necesario tener **Docker** instalado, o Docker Desktop si trabajas desde un ordenador Windows.

Copia las variables de entorno del archivo .env.example:
```
cp .env.dev.example .env
```

Para correr la api ejecuta el siguiente comando en el directorio raíz del proyecto:
```
docker compose -f docker-compose.yaml up -d --build
```

Si quieres parar y borrar los contenedores que están corriendo actualmente ejecuta:
```
docker compose -f docker-compose.yaml down
```

Activa la flag **-v** si quieres borrar los volúmenes para borrar los datos de la base de datos por ejemplo:
```
docker compose -f docker-compose.yaml down -v
```

Si quieres ver los contenedores corriendo ejecuta:
```
docker ps
```

Por último, si quieres ver los logs del contenedor del backend ejecuta:
```
docker logs -f backend_container
```

## Levantar el cliente

Para levantar la aplicación con el framework de Expo no se usarán contenedores, ya que esto dificuta bastante el montaje de la aplicación y es bastante fácil su uso en local. Por tanto se usará de manera local por simplicidad.

Situate en el directorio del frontend e instala el proyecto:
```
cd frontend
npm install
```

Posteriormente, se levantará el cliente con el siguiente comando:
```
npx expo start
```

Podrá acceder al cliente en el siguiente enlace desde el ordenador http://localhost:8081. Si quieres verlo en versión móvil abra el navegador, abra las herramientas de desarrollador y seleccione visualizar la web en versión movil.
