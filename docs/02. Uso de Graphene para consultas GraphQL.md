# Configuración de GraphQL

La configuración de GraphQL ya está lista. Puedes acceder a GraphiQL en: `http://localhost:8000/graphql/`

## Queries Disponibles

### 1. Obtener todos los elementos

```graphql
query {
  allMockElements {
    id
    name
    longitude
    latitude
    createdAt
  }
}
```

### 2. Obtener un elemento por ID

```graphql
query {
  mockElement(id: 1) {
    id
    name
    longitude
    latitude
    createdAt
  }
}
```

### 3. Buscar por name

```graphql
query {
  mockElementsByName(name: "Giralda") {
    id
    name
    longitude
    latitude
    createdAt
  }
}
```

## Estructura del Schema

- **MockElementType**: Tipo principal basado en el modelo MockElement
  - `id`: ID del elemento
  - `name`: Nombre del elemento
  - `longitude`: Longitud (coordenada X)
  - `latitude`: Latitud (coordenada Y)
  - `createdAt`: Fecha de creación
  - `puntoGeografico`: Punto geográfico completo

## Importante: Solo Queries (Read-Only)

**GraphQL en este proyecto está configurado SOLO para consultas (GET).**  
Las operaciones de modificación (INSERT, UPDATE, DELETE) se realizan directamente mediante consultas relacionales

**No se utilizan Mutations en GraphQL.**

## Para agregar más consultas

### Agregar queries para nuevos modelos

1. Define tu modelo en `main/models.py`
2. Crea un tipo GraphQL en `current/schema/types.py`:

```python
class NuevoModeloType(DjangoObjectType):
    class Meta:
        model = NuevoModelo
        fields = '__all__'  # o especifica los campos que quieres exponer
```

3. Añade queries en la clase `Query` del mismo archivo:

```python
class Query(graphene.ObjectType):
    # ... queries existentes ...

    # Nuevas queries
    all_nuevo_modelo = graphene.List(NuevoModeloType)
    nuevo_modelo_by_id = graphene.Field(NuevoModeloType, id=graphene.Int(required=True))

    def resolve_all_nuevo_modelo(self, info):
        return NuevoModelo.objects.all()

    def resolve_nuevo_modelo_by_id(self, info, id):
        try:
            return NuevoModelo.objects.get(pk=id)
        except NuevoModelo.DoesNotExist:
            return None
```

### Queries con filtros avanzados

```python
# En current/schema/types.py - dentro de la clase Query
elementos_cercanos = graphene.List(
    MockElementType,
    longitude=graphene.Float(required=True),
    latitude=graphene.Float(required=True),
    radio_km=graphene.Float(default_value=10.0)
)

def resolve_elementos_cercanos(self, info, longitude, latitude, radio_km):
    from django.contrib.gis.geos import Point
    from django.contrib.gis.measure import D

    punto = Point(longitude, latitude)
    return MockElement.objects.filter(
        punto_geografico__distance_lte=(punto, D(km=radio_km))
    )
```

### El schema se actualiza automáticamente

No necesitas modificar `current/schema/schema.py` - solo añades tipos y queries en `types.py`
