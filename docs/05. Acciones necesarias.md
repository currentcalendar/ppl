## Seguridad y Calidad de Código (DevSecOps)

Para fomentar la seguridad del código durante el desarrollo, el equipo de ciberseguridad ha propuesto un **hook de pre-commit**. Para instalarlo, ejecuta los siguientes comandos en la raíz del proyecto:

```
pip install pre-commit
pre-commit install
```
